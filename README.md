# 🖐️ R-Shiny AR 手势控制粒子系统 (Hand Gesture Particle AR)

> 一个基于 R Shiny 和 Google MediaPipe 的沉浸式增强现实 (AR) 实验。通过摄像头捕捉手势，实时控制全屏粒子物理系统。

------------------------------------------------------------------------

## 📖 项目简介

本项目是一个探索 **Web AR** 与 **数据科学框架 (R Shiny)** 结合的实验性作品。虽然宿主环境为 R 语言，但核心的视觉渲染与物理计算完全剥离至前端，实现了极高性能的交互体验。

用户只需开启摄像头，即可通过手势（张开或握拳）与屏幕上的粒子流进行零延迟的物理交互，体验“掌控星云”般的数字艺术效果。

## ✨ 核心亮点

### 🚀 零延迟物理引擎 (Zero-Latency Physics)

采用 **客户端渲染 (Client-side Rendering)** 架构。 \* **传统痛点解决**：虽然应用由 R 启动，但为了避免 R 单线程处理图形的性能瓶颈，所有粒子计算和渲染逻辑均由前端 JavaScript (Canvas API) 接管。 \* **性能表现**：在大多数设备上可实现稳定的 **60 FPS** 丝滑体验。

### 🤖 实时 AI 识别 (Real-time AI)

集成 **Google MediaPipe Hands** 模型。 \* **边缘计算**：直接在浏览器端进行手部骨骼关键点识别。 \* **隐私安全**：无需将视频流传输至后端服务器，保护用户隐私同时也极大降低了网络延迟。

### 🎨 沉浸式交互 (Immersive Interaction)

系统能精确识别手部状态并触发不同的物理场：

| 手势 | 状态 | 视觉效果 | 物理逻辑 |
|:-----------------|:-----------------|:-----------------|:-----------------|
| **✋ 张开手 (Open)** | **星云扩散** | 粒子呈青蓝色 (Cyan/Blue) | 产生随机漂浮力与斥力，粒子四散游动 |
| **✊ 握紧拳 (Fist)** | **能量聚合** | 粒子变火红色 (Fire Red) | 产生强力引力场，粒子被极速吸附至手心位置 |

### 🖥️ 全屏自适应

-   自动监听浏览器窗口大小变化 (Resize Event)。
-   移除所有 Shiny 默认的 UI 元素与边距，提供纯净的全屏数字艺术体验。

## 🛠️ 技术栈 (Tech Stack)

本项目采用 **前后端分离** 的设计思想：

-   **Backend (宿主环境)**
    -   🐶 **R**: 项目基础语言。
    -   💎 **Shiny**: Web 应用框架，负责路由与资源加载。
    -   🔌 **Shinyjs**: 用于 R 与 JavaScript 之间的通信（如需扩展后端控制逻辑）。
-   **Frontend (视觉与逻辑)**
    -   🎨 **HTML5 Canvas**: 高性能 2D 绘图上下文。
    -   ⚡ **JavaScript (ES6)**: 核心物理引擎编写与逻辑控制。
-   **AI Model**
    -   🧠 **MediaPipe Hands**: Google 开源的机器学习解决方案 (通过 JS CDN 引入)。

## 🏃 如何运行 (Quick Start)

确保你已经安装了 R 语言环境。

1.  **安装依赖包** `r     install.packages(c("shiny", "shinyjs"))`

2.  **克隆/下载项目** 将项目下载到本地目录。

3.  **启动应用** 在 R Studio 中打开 `app.R` (或 `server.R`/`ui.R`) 并点击 **Run App**，或者在控制台运行： `r     library(shiny)     runApp("path/to/your/project")`

4.  **授权摄像头** 浏览器弹出提示时，请允许使用摄像头权限以开启 AR 体验。
